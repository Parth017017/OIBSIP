## Project Title
Unemployment Analysis with Python

## Problem Statement

Unemployment is measured by the unemployment rate, which represents the percentage of people in the labor force who are unemployed.

During the COVID-19 pandemic, unemployment rates increased significantly across many regions. This project analyzes unemployment trends and identifies patterns using Python and data visualization techniques.

---

## Objective

The main objectives of this project are:

- Analyze unemployment data across different states/regions.
- Identify regions with high unemployment rates.
- Understand employment trends during the COVID-19 period.
- Perform exploratory data analysis (EDA).
- Visualize unemployment patterns using graphs.

---

## Dataset

Dataset Used:

**Unemployment_Rate_upto_11_2020.csv**

Dataset contains:

- Region (State)
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)
- Date Information
